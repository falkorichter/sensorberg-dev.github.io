async Task SensorbergSDK.SDKManager.InitializeAsync ( SdkConfiguration  configuration)

Initializes the SDK using the given configuration. The scanner can be used separately, but the resolving beacon actions cannot be done unless the SDK is initialized. If background task is enabled, this method check if there are updates for the background task filters available and updates them if so.

async Task SensorbergSDK.SDKManager.InitializeAsync ( string  apiKey,
string  timerClassName = null,
string  advertisementClassName = null,
string  uuidSpace = Constants.SensorbergUuidSpace,
bool  startScanning = true 
)

Initializes the SDK using the given API key. The scanner can be used separately, but the resolving beacon actions cannot be done unless the SDK is initialized. If background task is enabled, this method check if there are updates for the background task filters available and updates them if so.

Parameters
apiKeyThe API key for the Sensorberg service.
timerClassNameFull class name of the timer background process, if needed.
advertisementClassNameFull class name of the advertisement background process, if needed.
uuidSpaceUUID space for the background task, default value is Constants.SensorbergUuidSpace.
startScanningStart the background scanner.
